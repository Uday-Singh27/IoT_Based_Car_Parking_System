# IoT_Based_Car_Parking_System
IoT Based Car Parking System: A smart parking solution that leverages Arduino and ESP8266 Wi-Fi Module to automate parking slot detection and gate control in real-time. Designed to eliminate manual effort and reduce vehicle waiting time in congested parking areas.
Features:

Real-time parking slot availability monitoring
Automated gate control based on slot occupancy
Seamless IoT connectivity via ESP8266
100% reduction in manual intervention

Tech Stack:
Arduino ,ESP8266, IoT, IR Sensors ,Servo Motor,Arduino IDE

Team:
Built and deployed by a team of 4 — covering circuit design, sensor integration, hardware assembly, testing & debugging.
